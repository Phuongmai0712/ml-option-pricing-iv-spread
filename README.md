# Machine Learning Applications in Option Pricing, Implied Volatility, and Bid-Ask Spread Prediction

This project applies machine learning models to three option-related tasks:

1. Synthetic American call option pricing
2. Implied volatility prediction
3. Bid-ask spread prediction

## Models used
- Linear Regression
- SVR
- Decision Tree
- Random Forest
- Gradient Boosting
- XGBoost
- LightGBM
- Neural Network

## Evaluation metrics
- MSE
- RMSE
- MAE
- R²

## Main findings
- Neural Network performed best on synthetic pricing
- Random Forest performed best on implied volatility
- XGBoost performed best on bid-ask spread prediction
